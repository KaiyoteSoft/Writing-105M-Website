Stanford Electrical Engineering

Development for Stanford's EE and CS staff

I interned at Stanford over the summer, building various programs and websites under the instruction of Joeseph Little. Unfortunately, much of the work I did is Stanford property, and therefore protected. However, you can see an example of a organizational program that I made for the Computer Science staff. This program receives the data from hundreds of computers in JSON format. The program then parses the data by value (hardware address, ip address or name) and displays the results to the screen in a table. From there, subsearches on other values are made possible. To get started, click the button and type "lion" into the first input box.

Reslet Summer of API's

Delving into design

Restlet is a company that offers API hosting services. I thought it would be interesting to test out their product while focusing more on the desgin aspect of a webpage. The functionality of this project is very similar to the Stanford program above, but the look and feel of the page is much different. By combining Bootstrap and Jquery I was able to create a clean design that pulls the data from a third party source. Additionally there is a greater focus on the theme of the page and the look of certain elements.
